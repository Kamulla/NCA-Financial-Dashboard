## Overview
Interactive dashboard for **National Construction Authority (NCA)** financial performance data covering **FY 2012/2013 – 2024/2025**.  

The dashboard provides **board-level insights** into the organization’s financials, allowing stakeholders to easily explore and analyze income, expenditure, assets, and liabilities.  

**Features:**
- **Key KPIs**: Total Income, Expenditure, Assets, Liabilities, and Ratios  
- **Overview Tab**: High-level summary with interactive charts  
- **Detailed Tabs**: Income, Expenditure, Assets, Liabilities  
- **Visualizations**: Trend lines, treemaps, pie charts, bubble charts  
- **Filters**: View by single year or all years combined  
- **Ratios**:  
  - **Asset-to-Liability Ratio**  
  - **Income-to-Expenditure Ratio** 

## Usage
- Run the dashboard locally with Streamlit or view online.  
- Use the sidebar to filter by year.
