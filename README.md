# NCA 10-Year Financial Performance Dashboard

## Overview
Interactive dashboard for **National Construction Authority (NCA)** financial performance data covering FY 2012/13 – 2024/25.  

**Features:**
- Key KPIs: Income, Expenditure, Assets, Liabilities, and Ratios  
- Tabs for Overview, Income, Expenditure, Assets, Liabilities  
- Trend analysis, treemaps, and bubble charts  
- Filters by financial year or all years combined  

## Usage
- Run the dashboard locally with Streamlit or view online.  
- Use the sidebar to filter by year.  

## Deployment
Hosted on **Streamlit Cloud**: [View Dashboard](https://nca-financial-dashboard-c5dpy5rou7x3uudrnfjbyj.streamlit.app/)
## Technologies Used
- Python, Streamlit, Pandas, Plotly, openpyxl
